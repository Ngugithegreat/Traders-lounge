# **App Name**: Traders Lounge

## Core Features:

- Visual Bot Builder: An intuitive drag-and-drop interface that allows users to construct automated trading logic and strategies without coding.
- AI Market Signal Tool: Utilizes a reasoning tool to evaluate live synthetic index data and decide the most profitable entry points for automated execution.
- Live Performance Dashboard: Real-time visualization of trading sessions, including P&L tracking, balance updates, and active bot status.
- Free Strategy Library: A repository of pre-configured trading bots and strategies for indices like Boom, Crash, and Volatility.
- Secure Deriv Integration: Seamless OAuth2 connection to Deriv accounts for real-money and demo account synchronization.
- Social Copy Trading: System to follow and automatically mirror the trades of high-performing users within the community.

## Style Guidelines:

- Primary Color: Electric Neon Green (hsl(158 100% 44%)) to signify growth, profitability, and positive action.
- Background Color: Deep Charcoal Blue (hsl(220 30% 7%)) to provide a high-contrast, professional trading environment.
- Headlines: 'Space Grotesk' font for a modern, futuristic, and high-tech aesthetic.
- Body Text: 'Inter' font to ensure maximum legibility for dense financial data and complex charts.
- Glassmorphism components with subtle blurs and thin borders to create a layered, multi-dimensional dashboard feel.
- Minimalist and sharp icons combined with specific emoji identifiers for distinct trading tools and index types.
- Dynamic scrolling tickers for real-time market prices and subtle pulse animations for active trading indicators.
